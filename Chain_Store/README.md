# JavaFX Chain Store Management System

This project is a **Chain Store Management System** developed using **Java** and **JavaFX** within the scope of the *Object-Oriented Programming* course.

The system models the core operations of a retail chain with multiple branches, including product management, stock tracking, and sales operations, following object-oriented design principles.
